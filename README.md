# GreySkies

A Unity project featuring custom shader development and graphics implementation.

## Overview

GreySkies is a shader-focused project built with Unity, demonstrating advanced graphics programming techniques and shader implementations using ShaderLab.

## Project Structure

```
GreySkies/
├── Assets/              # Game assets including models, textures, and scenes
├── Packages/            # Unity packages and dependencies
├── ProjectSettings/     # Unity project configuration
├── .vscode/            # VS Code configuration for the project
└── GreySkies.slnx      # Solution file for the project
```

## Requirements

- **Unity** (version to be specified)
- **Visual Studio Code** (recommended for development)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/FullMoonie2719/GreySkies.git
   cd GreySkies
   ```

2. **Open in Unity:**
   - Launch Unity Hub
   - Add the project from the cloned directory
   - Open the project

3. **Development:**
   - Edit shaders and assets in the `Assets/` folder
   - Configure project settings in `ProjectSettings/`
